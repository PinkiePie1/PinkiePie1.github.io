作业：目标用户，与创造性的问题（7.9更新）

发表于 2017-07-06 | 分类于 作业

目标

  • 将田野调查和案头调研的核心数据移到Post-it上

  • 将数据进行分类组合，找出pattern和theme

  • 总结出目标用户的User Persona

  • 找出目标用户的需求和洞察

  • 将用户，需求和洞察结合形成HMW Question

User Persona

“比起使用现金购买商品，我更喜欢用支付宝支付。不过，坦白地讲，我会选择直接在网上购物，而不是去商场，因为这真的很方便。我和我的朋友们会在微信上聊天，分享我的日常生活。我无法想象如果有一天没有智能手机和互联网，该如何生活下去。”

基本信息：

  • 年龄：40~55
  • 职业：未知 / 退休
  • 地点：中国上海

特征信息：

  • 拥有智能手机、电脑，家中有联网设备、WIFI。
  • 会基础的产品操作（如手机的开关机、打电话、发短信、下载应用程序，访问网页等等）。
  • 了解微信、支付宝。其基本功能（如微信的聊天，支付宝的付款）已使生活收益。
  • 了解目前手机与电脑、互联网功能的强大。
  • 对利用智能手机应用、电脑，使生活更幸福（便捷、娱乐）有浓厚兴趣。
  • 日常生活中使用以上产品较为频繁。

Insight

从针对老年人的调查中……

  • 老年人受时代影响，多数已学会使用手机基本功能。微信普及度也十分广泛，老年人会使用微信建立自己的群并参与聊天。
  • 老年人对涉及到金钱的项目具有很强的安全意识，也因此拒绝使用如微信钱包、支付宝等便民应用，认为其“不安全”。
  • 对居委会中采访的老年人，普遍认为生活已足够幸福。拥有丰富的日常活动，无忧无虑。并习惯保持他们的节奏生活。
  • 对手机、电脑了解片面，信息量不足。怕弄坏，也因不知道去哪里修理而嫌麻烦。

从针对未来老年人的采访与观察中……

  • 习惯使用智能手机与电脑，对电子科技产品有一定的关注度（iphone）。
  • 习惯使用非现金支付方式。
  • 对电子科技产品的应用功能了解不够完全。相比手机，对电脑功能更好的应用需要指导。
  • 对通过电子科技产品使生活更幸福有兴趣。

总结

我们发现，受时代影响，未来的老年人普遍拥有并使用智能手机与电脑，利用互联网功能带来便捷，手机与电脑已与他们的生活分不开。同时，他们对这些电子科技产品上应用程序已有所了解，并有使用经验，即对其实用性、安全性有了大致的了解。

但仍存在的问题是，他们并不能完全掌握电子科技产品的应用功能。随着版本的更新。他们可能仍停留在使用基础功能，对于新增加的功能/未发现的功能并无了解。另外，由于日常生活中接触手机较多，他们对电脑的理解远不如手机出色，电脑与手机对未来的老年人有同样的挖掘价值。凭借未来的老年人已有的使用经验，该人群拥有学习更好地使用电子科技产品的潜力。

从老年人的调查中发现，老年人的智能手机通常是由“未来的老年人”一辈所购买。再从自身的经历与推理可得，“未来的老年人”对其子女也有着电子科技产品启蒙的作用。

“未来的老年人”在通过使用电子科技产品提高自己生活质量的同时，也可兼顾到“老年人”，用便捷的方式提供老年人所需求的服务。我们有理由相信，更好地使用电子科技产品能够使生活更幸福。

HMW Question

我们如何帮助即将在十年之内退休的下一代老年人充分的利用当代流行的，便民的手机应用程序使生活更加便利？
